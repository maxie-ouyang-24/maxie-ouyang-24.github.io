CSS Concepts

Display Inline vs Inline Blocks

You are creating a webpage and you've added two boxes. You'd like for them to be displayed side by side instead of one on top of the other. To do this, you use the display property.

Default - Display Block

By default, they look like this.

Code: display: block;

Display Inline

Let's say you'd like them to be side by side from each other.

Code: display: inline;

Display Inline-Block

Let's say you'd like them to be side by side from each other.

Code: display: inline-block;
